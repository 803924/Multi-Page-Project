Cross Country

Source: usatf.org


Cross country is a running sport in which teams and individuals

run races over natural terrain. Races usually are run over courses

confined country, fields, parks, golf courses, and grasslands. paths over

of paved roads should be kept to a minimum. Races are mainly 5k (5000 meters)

which is 3.1 miles.
